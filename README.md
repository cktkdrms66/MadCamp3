## 기능 개요

dlflwjfl는 자이로센서를 기반으로 제작된 폭탄 돌리기 게임입니다.

- 최대 20명까지 참여 가능한 방을 만들 수 있습니다. 모두 같이 플레이하세요!
- Android iOS 모두 지원합니다! 
- 좀비가 되면 이름표가 빨간색이 됩니다. 제한 시간 내에 친구들을 잡아 좀비 상태에서 벗어나세요.
